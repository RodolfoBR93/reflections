1) Quando gostaria de usar um repositório remoto em vez de manter o trabalho local?

R) Sempre que tiver um projeto grande é ou precisar fazer algumas mudanças que não sabe se dará certo é recomendado utilizar um repositório remoto

2)por que você prefere efetuar pull de alterações manualmente do que deixar o Git atualizar-se automaticamente com o seu repositório remoto?

R)Se o git fizesse pull(puxar) as alterações automaticamnte, poderia ocorrer conflitos nos arquivos, eu poderia estar modificando um arquivo no momento e isso não seria legal

3)Descreva as diferenças entre bifurcações, clones e branches. Quando você usaria um e não os outros?

R) Bifurcação é quando eu copio o repositório de alguém no guithub para meu github.
clone é quando baixo o repositório de alguém do github para meu computador.
branches são as versões diferentes do meu projeto.

4) Qual é o benefício de ter uma cópia do último estado conhecido do remoto armazenado localmente?

R)poder recuperar os arquivos se algo der errado
